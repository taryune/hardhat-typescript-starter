# hardhat-typescript-starter
Uses  
- [Hardhat](https://github.com/nomiclabs/hardhat)
- [Ethers](https://github.com/ethers-io/ethers.js/)
- [Waffle](https://github.com/EthWorks/Waffle)

## Usage
### Install dependencies
```
yarn
```

### Build
```
yarn build
```

### Test
```
yarn test
```


### Lint & Prettier
```
yarn fix
```
